Database Design for Mastermind Plumbing and Maintenance

Customers Table

Field Type
CustomerID INT (Primary Key, Auto Increment)
Name VARCHAR(255)
Phone VARCHAR(20)
Email VARCHAR(255)
Address VARCHAR(255)

Services Table

Field Type
ServiceID INT (Primary Key, Auto Increment)
ServiceName VARCHAR(255)
Description TEXT
Price DECIMAL(10, 2)

Payments Table

Field Type
PaymentID INT (Primary Key, Auto Increment)
AppointmentID INT (Foreign Key - References Customers Table CustomerID)
Amount DECIMAL(10, 2)
PaymentDate DATE
PaymentMethod VARCHAR(50)

Admin Table

Field Type
AdminID INT (Primary Key, Auto Increment)
Username VARCHAR(255)
Password VARCHAR(255)
Email VARCHAR(255)
Role VARCHAR(50)

PayPal Gateway Table

Field Type
GatewayID INT (Primary Key, Auto Increment)
MerchantID VARCHAR(255)
APIUsername VARCHAR(255)
APIPassword VARCHAR(255)
Signature VARCHAR(255)

Contact Us Table

Field Type
ContactID INT (Primary Key, Auto Increment)
Name VARCHAR(255)
Email VARCHAR(255)
Message TEXT

Service Team Table

Field Type
TeamID INT (Primary Key, Auto Increment)
Name VARCHAR(255)
Phone VARCHAR(20)
Email VARCHAR(255)
Position VARCHAR(50)
